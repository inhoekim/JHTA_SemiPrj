<h3>개별(우리우리)호텔 웹사이트 개발 프로젝트</h3>

- 웹사이트를 이용하는 사용자들에게 호텔에 대한 정보를 제공하는 사이트입니다

- 사용자에게 달력API를 제공하여 현재 이용할 수 있는 객실을 검색할 수 있게하고, 추가적으로 예약까지 진행할 수 있도록 도와줍니다

- 사용자는 자신이 이용했던 호텔 객실에 대한 한줄평을 사이트에서 남길 수 있습니다
